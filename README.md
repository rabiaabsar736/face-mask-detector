# Face Mask Detection 😷

This project is a Deep Learning web app that detects whether a person is wearing a face mask or not.

## 🚀 Tech Stack
- TensorFlow / Keras
- Gradio
- Hugging Face Spaces
- Kaggle API

## 🌐 Live Demo
https://huggingface.co/spaces/RabiaAbsar/face-mask-detector

## 📌 Features
- Upload image
- Detect mask / no mask
- Shows prediction confidence

## 🧠 Learning Highlights
- CNN for image classification
- Kaggle dataset handling
- Overfitting prevention using Dropout
- Model deployment using Gradio

- app.py
